# Dema.ai Test
## Run app locally
- `npm install`
- `npm run dev`
## Unit test
- `npm run test`
## Component test (using Cypress)
- `npm run dev`
- Open another terminal `npm run cypress`
- Choose "Component Testing" option 
## E2E test (using Cypress)
- `npm run dev`
- Open another terminal `npm run cypress`
- Choose "E2E Testing" option
